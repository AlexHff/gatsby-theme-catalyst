# Gatsby-Theme-Catalyst-Writer

**Demos:**

- [gatsby-starter-catalyst-writer](https://gatsby-starter-catalyst-basic.netlify.com/)
- [www.briannasharpe.com](https://www.briannasharpe.com)

This is a full-featured theme that ties together the core theme, header and footer themes and then uses this as a basis to create a full website. The theme relies on data from SANITY.io and includes a customized content studio for the CMS.
